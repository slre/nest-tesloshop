#Teslo API

1.- Clone Project

2.- Install Dependencies 
```
npm install
```

3.- Clone and rename file  
````
env.template
````
4.-Setting up environment variables

5.-Start and connect the database [docker]

```
docker-compose up -d
```
6.-Run Server ( check 'scripts' in  package.json  )
```
npm start --watch
```
